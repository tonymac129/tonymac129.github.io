# [TonyMac129](https://tonymac.net)

This is my portfolio and personal website with my featured projects, medium-scale projects, misc projects, skills, blog posts, about, contact, and other information related to me. View the site at [tonymac.net](https://tonymac.net) now!

## Pages

- Featured: my best projects with detailed info and link
- Projects: my medium-scale projects with tech stack and short description
- Misc: my very small, abandoned, learning, or non-coding related projects
- Skills: my coding, tech, and general skills listed
- Blog: where I share about my skills, ideas, or just random stuff
- About: general information about me things I use and like, and contact information

## Features

- Powerful search across all my projects and posts
- Dark mode toggle
- Animation for smooth experience
- Client side rendering for fast page loads
- Toggle between grid and gallery views to see all my projects
- Interactive greeting message on the homepage

## Tech Stack

- Core: React and CSS
- Libraries: Framer Motion for animation and React Router for routing
- Deployment: Vite for dev and build and GitHub Pages for deployment
